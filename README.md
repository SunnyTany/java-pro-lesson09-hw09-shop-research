# 📊 Shop Research Data Handler

Приложение для обработки данных о ежедневных продажах магазина. 

Проект демонстрирует использование Generic-методов для работы с различными типами данных и эффективное манипулирование строками.

## 🚀 Особенности проекта

- Универсальность (Generics): Класс DataHandler содержит метод handleData, который одинаково успешно обрабатывает массивы строк (названия товаров) и массивы чисел (суммы продаж).
- Оптимизация StringBuilder: Удаление лишнего пробела в конце строки реализовано через deleteCharAt, что эффективнее по памяти, чем использование метода .trim().
- Чистый код: Использование цикла for-each и форматированного вывода String.format делает код легкочитаемым.

## 🛠 Технологический стек

- Java: Использование обобщенных типов данных (Type Parameters).
- StringBuilder: Для высокопроизводительной сборки строк.

## 📂 Структура кода

Центральным элементом является метод handleData:

    public <T> String handleData(T[] items) {
        
        StringBuilder sb = new StringBuilder();
        
        int count = 0;
        
        for (T item : items) {
            
            count++;
            
            sb.append(String.format("(%d) %s ", count, item));
        
        }
        
        // Эффективное удаление последнего пробела
        
        if (sb.length() > 0) {
            
            sb.deleteCharAt(sb.length() - 1);
        
        }
        
        return sb.toString();

    }

## 💻 Пример работы программы

Программа преобразует массивы данных в читаемый список:

Вход: ["apple", "grape", "mango"]

Выход: Products: (1) apple (2) grape (3) mango

Вход: [1520.89, 2058.35]

Выход: Sales, EUR: (1) 1520.89 (2) 2058.35
